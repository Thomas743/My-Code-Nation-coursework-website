GDPR will restrict what access I have to a users information and how it is stored. If I had a login/registration system on my website I will have to be clear on what information I stored and get the users consent. If I have a data breach on my website I should contact users within 3 days notifying them of the situation and show them what information about them my website had. I will also have to make sure my website has good security on the website which will add time to the development of the website.
